## 📌 Overview

AI-powered travel planner that generates personalized itineraries using **LangChain Agents**.
The system dynamically decides when to use tools like **place search, budget calculator, and weather API**.

---

## ✨ Features

* 📅 Day-wise itinerary generation
* 💰 Budget breakdown (realistic pricing)
* 🌦 Weather-aware recommendations
* 🗺 Google Maps integration
* 💡 Smart travel tips
* 🧠 Agent-based reasoning (ReAct framework)
* 🎨 Clean card-based UI (Streamlit)

---

## 🧠 How It Works

User Query → Agent → Tool Selection → Data Retrieval → Final Plan

* **Agent:** decides what action to take
* **Tools:** database, budget, weather
* **LLM:** reasoning + response generation

---

## 🛠 Tech Stack
